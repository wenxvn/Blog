## 题目链接

[P1001 A+B Problem](https://www.luogu.com.cn/problem/P1001)

## 思路讲解

读取两个整数 `a` 和 `b`，输出它们的和。该题主要用于熟悉输入输出格式。

## C++ 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    long long a, b;
    cin >> a >> b;
    cout << a + b << '\n';
    return 0;
}
```
