非常简单的一题


## 代码
```c++
#include<bits/stdc++.h>
using namespace std;
int F;
int main(){
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	cin>>F;
	int c=5*(F-32)/9;
	cout<<"Celsius = "<<c;
	return 0;
}
```
