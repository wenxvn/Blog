## 思路

用a,b,c,d四个字母模拟分数相加

每次加完都要约分（求最大公因数）

最后分类讨论各种输出情况

## 注意
如何读取分数？

我感觉用scanf("%d/%d",&c,&d);是最合适的

但是，如果前面写了关闭流同步，用了scanf之后就不能再用cin来读取n了，否则提交时会出现“浮点错误（您的程序运行时发生浮点错误，比如遇到了除以 0 的情况）”的报错

## 代码
```c++
#include<bits/stdc++.h>
using namespace std;
int n;
int gcd(int x,int y){
	if(x>y)swap(x,y);
	while(x){
		int t=y%x;
		y=x;
		x=t;
	}
	return y;
}
int main(){
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	scanf("%d",&n);
	int a=1;//分子 
	int b=0;//分母 
	while(n--){
		int c,d;
		scanf("%d/%d",&c,&d);
		b=b*d+c*a;
		a=a*d;
		int k=gcd(a,b);
		a/=k;
		b/=k; 
	}
	if(b==0)printf("0");
	else{
		if(b%a==0)printf("%d",b/a);
		else {
			if(b>a)printf("%d %d/%d",b/a,b%a,a);
			else printf("%d/%d",b,a);
		}
	}
	return 0;
}
```
