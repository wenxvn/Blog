## 思路

按照题意写即可，我个人喜欢用printf来处理需要限制小数位的输出

printf("%.xf")，x就表示需要保留到小数点后几位

## 代码
```c++
#include<bits/stdc++.h>
using namespace std;
string s;
double sum,two;
int main(){
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	cin>>s;
	for(int i=0;i<s.size();i++){
		if(s[i]=='-')continue;
		sum+=1;
		if((s[i]-'0')==2)two+=1;
	} 
	two*=100;
	if(s[0]=='-')two*=1.5;
	if(s[s.size()-1]%2==0)two*=2;
	printf("%.2f%%",two/sum);
	return 0;
}
```
