## 思路
要求四舍五入,不需要写if-else判断小数部分,只要(n+1)/2

求n除以x的向上取整,公式为(n+x-1)/x


## 代码

```c++
#include<bits/stdc++.h>
using namespace std;
int main(){
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	int n;
	char c;
	cin>>n>>c;
	int k=(n+1)/2;
	for(int i=0;i<k;i++){
		for(int j=0;j<n;j++){
			cout<<c;
		}
		cout<<'\n';
	}
	return 0;
}
```
