## 思路
用getline(cin,s)直接读入一整行（包括空格）

用哈希表标记需要删除的字符

当然，更简单的方法是直接开一个256大小的数组来存储字符

## 为什么不用写cin.ignore()
cin.ignore()用来处理cin之后的换行符，如果上面代码是用cin来处理输入，下面就要用cin.ignore()

getline()会自行处理换行符，不需要手动处理

## 代码
```c++
#include<bits/stdc++.h>
using namespace std;
string s,t;
int main(){
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	getline(cin,s);
	getline(cin,t);
	unordered_map<char,int>map;
	for(char c:t)map[c]++;
	for(char c:s){
		if(map[c])continue;
		cout<<c;
	}
	return 0;
}
```
