## 思路
需要掌握两个处理字符串的函数

string t=s.substr(index,len);可以切割字符串,index为起始位置,len为切割的长度

int n=stoi(t);可以将string类型的数字转化为int类型

## 代码
```c++
#include <bits/stdc++.h>
using namespace std;
int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    string time;cin >> time;
    int hour = stoi(time.substr(0, 2));
    int min = stoi(time.substr(3, 2));
    if (hour >= 0 && hour <= 11 || hour == 12 && min == 0)
    {
        cout << "Only " << time << ".  Too early to Dang.";
        return 0;
    }
    if (min)hour++;
    hour -= 12;
    while (hour--)cout << "Dang";
    return 0;
}
```
