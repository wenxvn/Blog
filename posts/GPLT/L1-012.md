## 思路
直接用pow()函数
pow(x,y)表示x的y次方

## 代码
```c++
#include<bits/stdc++.h>
using namespace std;
int main(){
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	int n;cin>>n;
	printf("2^%d = %d",n,(int)pow(2,n)); 
	return 0;
}
```
