## 思路
查找和特定数字绑定的其他元素，可以用哈希表实现，key是待查找的数字，value是需要输出的其他元素

## 代码
```c++
#include<bits/stdc++.h>
using namespace std;
int n;
int main(){
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	cin>>n;
	unordered_map<int,pair<string,int>>map;
	string s;int a,b;
	while(n--){
		cin>>s>>a>>b;
		map[a]={s,b};
	}
	cin>>n;
	int t;
	while(n--){
		cin>>t;
		cout<<map[t].first<<" "<<map[t].second<<'\n';
	}
	return 0;
}
```
