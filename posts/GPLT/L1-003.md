## 我的思路
用哈希表记录数字的出现次数

由于最后还要按数字的升序输出，所以还要放进vector中sort

可以AC

## 代码
```c++
#include<bits/stdc++.h>
using namespace std;
string n;
bool com(const pair<int,int>&x,const pair<int,int>&y){
	return x.first<y.first;
}
int main(){
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	cin>>n;
	unordered_map<int,int>map;
	for(char c:n)map[c-'0']++;
	vector<pair<int,int>>ans;
	for(auto &t:map){
		ans.push_back({t.first,t.second});
	}
	sort(ans.begin(),ans.end(),com);
	for(int i=0;i<ans.size();i++){
		cout<<ans[i].first<<":"<<ans[i].second<<'\n';
	}
	return 0;
}
```

## 简化思路

上面其实想复杂了

数字0~9一共只有10种类型，可以直接开大小为10的数组来存

最后也不需要sort，直接输出>0的记录即可

## 简化版代码

```c++
#include<bits/stdc++.h>
using namespace std;
string n;
int cnt[10];
int main(){
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	cin>>n;
	for(char c:n)cnt[c-'0']++;
	for(int i=0;i<10;i++){
		if(cnt[i])printf("%d:%d\n",i,cnt[i]);
	}
	return 0;
}
```
